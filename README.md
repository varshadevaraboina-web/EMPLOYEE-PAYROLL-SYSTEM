 **Employee Payroll System**
 This project is a simple yet functional **Employee Payroll System** designed to manage employee records,
 calculate salaries, and generate payslips. It streamlines payroll operations for small to medium-sized 
 organizations by automating salary computation and record keeping.


 **Purpose**
Manual payroll management is time-consuming and error-prone. This system automates key payroll tasks such as:
- Storing employee details
- Calculating net salary based on attendance, deductions, and bonuses
- Generating payslips
- Maintaining salary history



**Features**
- Add, update, and delete employee records  
- Calculate gross and net salary  
- Apply deductions and bonuses  
- Generate printable payslips  
- View salary history and reports

 **How It Works**
1. Admin logs in to the system  
2. Adds employee details (name, ID, designation, salary)  
3. Inputs attendance, deductions, and bonuses  
4. System calculates net salary  
5. Payslip is generated and stored

**Use Cases**
- HR departments in small companies  
- Freelancers managing payroll for clients  
- Educational projects on database and UI integration
